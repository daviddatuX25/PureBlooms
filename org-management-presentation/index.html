Gabay sa Paggamit ng Portal

"Ito ang visual slides na ipo-project sa screen habang nagpe-present."

Para saan ito? Ang subpage na ito ay naglalaman ng slides deck na ipapakita sa projector ng klase. Naka-style ito gamit ang madilim na tema at custom animations para makuha natin ang visual excellence na hahangaan ng mga panelist.

Anong makikita rito? Naka-embed dito ang ating OM Core frameworks, workflow transitions, at system architecture diagrams na hindi mawawala sa ayos kahit anong screen size.

"Eto naman ang cockpit panel para sa mismong presenters para iwas-kaba habang nagsasalita."

Para saan ito? Ang screen na ito ay ginagamit ng mga speakers habang nagpe-present. Nilagyan natin ito ng live visual controls gaya ng timers at cues.

Anong makikita rito? Mayroon itong listahan ng acts ng script, at may buttons para i-filter ang lines ng bawat tao (mula kay David hanggang kay Novel) para malinaw kung sino ang susunod na magsasalita.

"Dito pwedeng i-verify agad sa browser ang buong details ng system report natin."

Para saan ito? Isang mabilisang preview ng dokumentasyon na isusumite sa instructor. Naka-format ito gamit ang Markdown para madaling basahin nang direkta sa browser nang walang kailangang i-download.

Anong makikita rito? Makikita dito ang operational functions, system specification at code analysis, database schema rules, at listahan ng collaboration contributions ng bawat miyembro.

"Ito ang soft-copy report na gagamitan ng official academic layout na isusumite sa instructor."

Para saan ito? Ito ang Word file (.docx) na idownload para sa opisyal na pagpasa ng developed system requirement.

⚠️ MAHALAGANG PAALALA: Ang file na ito ay kasalukuyang nasa draft status. Bago i-email o i-print para isumite sa prof, kailangan niyo munang buksan ang Word at ilagay ang actual system screenshots sa mga tagged spaces, at i-edit ang pangalan at iba pang cover details sa unang pahina.
